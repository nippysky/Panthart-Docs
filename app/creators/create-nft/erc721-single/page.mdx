# Single NFT (ERC-721)

**Best for:** a true 1/1 with unique metadata.

<Callout type="info">
Provide a **preview image** for fast loading and include full-res media via `image` or `animation_url` on IPFS.
</Callout>

<Steps>

### Prepare metadata
`name`, `description`, `image` (or `animation_url`), `attributes`.

### Creation
Pay the **creation fee** (covers IPFS + service). Confirm wallet prompts.

### Mint
Confirm mint to your wallet; provenance begins at block inclusion.

### Sell
Create a **fixed-price** listing or start an **auction**.

</Steps>

<Callout type="warning">
Edits after reveal can harm collector trust. Treat metadata as **immutable** after listing.
</Callout>
