# Quickstart

<Callout type="info">
Panthart runs on **Electroneum EVM**. Make sure your wallet is on the correct network.
</Callout>

<Steps>

### Install a wallet
Use **MetaMask** (or compatible EVM wallet).

### Connect
Open Panthart, click **Connect Wallet**, approve in your wallet.

### Try a read-only action
Open any collection or item page to validate network & indexing.

### Mint (optional)
Follow **Create** to mint: 721 Drop / Single 721 / Single 1155.  
Creation includes a fee that covers **IPFS + service**, plus normal gas.

### List for sale
Set fixed price or start an auction; approve marketplace (first time).

</Steps>
