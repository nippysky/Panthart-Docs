# Single (ERC-1155)

**Best for:** editions/passes with shared metadata.

<Steps>

### Prepare shared metadata
Include a clear title and edition context (e.g., “Pass v1”).

### Set supply
Choose total editions. Scarcity is a creative decision; you can mint initial supply now and enable public mint later.

### Creation
Pay **creation fee** (IPFS + service).

### Mint or enable public mint
Mint to treasury/creator wallet or open public mint.

### List
Sell editions individually or as a lot.

</Steps>

<Callout type="info">
All holders share the same metadata. If per-edition serials matter, surface them via transfer history or an indexer.
</Callout>
